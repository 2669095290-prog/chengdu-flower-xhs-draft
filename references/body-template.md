# Body Template

## Structure

Use short, scannable paragraphs. Avoid formal advertising language.

```text
成都同城订花，今天这束{色系/风格}花束很适合{使用场景/送人对象}。

不是{常见踩雷选择}，
是{颜色、花材、氛围}混在一起的{情绪词}。
一眼看过去会觉得：{用户会产生的感受}。

这束比较适合：
生日花束 / 见女朋友 / 闺蜜下午茶 / 纪念日 / 拍照出片。

成都这边很多客人订花都会先问：
今天能不能送？预算多少合适？送女生会不会踩雷？

如果你也不知道怎么选，
可以直接发我：预算 + 区域 + 送花时间 + 送谁。
我按对象和场景搭，不让送花变成开盲盒。

成都同城可自取/配送，急单先问时间。

#{城市花店} #{城市订花} #{城市同城鲜花} #{城市生日花束} #{送女友花束} #{色系花束} #{花束定制}
```

## Local Purchase Habits To Reflect

Chengdu flower-shop users often care about:

- Today delivery: `今天能不能送`, `急单先问时间`.
- Budget fit: ask for budget before recommending.
- Area/logistics: ask for district or delivery area.
- Recipient and scene: girlfriend, birthday, anniversary, best friend, afternoon tea, photo-taking.
- Anti-risk: avoid "踩雷", "开盲盒", and generic red roses when the bouquet has a more specific style.

## Style Rules

- Use concrete details from the image.
- Keep claims believable; do not invent exact prices, sales volume, or delivery guarantees.
- Use `可以直接发我：预算 + 区域 + 送花时间 + 送谁。` as the default conversion line.
- Use 6-10 visible hashtags. Put the strongest local tags first.
